// Designing the next reality.

C-CUBE

A platform of entrepreneurship education for college students, initiating their paths to start-up.

Medium Web Design / UI/UX Design
Date 2023 – 2024
Role Lead Designer & Frontend Developer

Web Design

Landing Pages

C-Cube delivers an easy-to-read user interface that creates a straightforward user experience. With clear visual hierarchy and thoughtfully designed layouts, every student entrepreneur can quickly find the resources they need.

The homepage features large visual heroes paired with refined information zones, guiding users to intuitively explore courses, events, and startup resources.

User Research

Deeply Understand User Scenarios

Through in-depth user interviews and scenario analysis, we uncovered the challenges and needs college student entrepreneurs face when seeking resources, building networks, and participating in events.

These research insights directly drove the design of the platform's core features, ensuring every module precisely addresses real user needs.

Grid Showcase

"Aim To Develop A Station To Assist Student Entrepreneurs"

Development

Build The Website From Scratch

The entire C-Cube website was designed and built from scratch by me, encompassing layout planning, visual design systems, frontend code implementation, and responsive layout design across all pages.

This process gave me a profound understanding of the tight relationship between design and development, and how to transform visual concepts into functional digital products.

Information Design

C-Cube Information Architecture

We carefully mapped the entire platform's information architecture, ensuring users can find all features and content through intuitive navigation paths.

A clear sitemap helped the team align on a shared understanding of the product, and served as a crucial reference blueprint for frontend development.
